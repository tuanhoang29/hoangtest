üìç How to Find Hotspot Coordinates:

Open your VR360 tour in a web browser (index.html)
Navigate to the scene where you want to add a hotspot
Press F12 or right-click and select "Inspect" to open Developer Tools
Go to the Console tab
Copy and paste the code below into the console, then press Enter
Move your mouse around the VR scene - coordinates will appear in the console
Click where you want the hotspot when you find the perfect position
Copy the yaw and pitch values from the console output

üí° Pro Tips:

  • Yaw range: -3.14 (far left) to +3.14 (far right)
  • Pitch range: -1.57 (straight down) to +1.57 (straight up)
  • Center: Yaw: 0, Pitch: 0
  • Test coordinates: Use the Hotspot Editor to preview before adding to data.js
  • Multiple attempts: Take several readings and average them for precision
